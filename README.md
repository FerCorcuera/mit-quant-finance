# MIT Quant Finance

This repository contains my notes, exercises, simulations, and programming practice based on:

- **MITx MicroMasters – Mathematical Methods for Quantitative Finance**
- **MIT OpenCourseWare – Topics in Mathematics with Applications in Finance**
- Additional material from:
  - *Probabilistic Machine Learning* (O'Reilly)
  - Quantitative Finance textbooks and lecture notes

The goal of this repository is not only to study the mathematical foundations of quantitative finance, but also to strengthen my programming skills by implementing concepts in both **R** and **Python**.

---

# Topics Covered

- Probability
- Statistics
- Linear Algebra
- Calculus
- Stochastic Processes
- Quantitative Finance
- Machine Learning for Finance

# Learning Philosophy

For each concept I try to:

1. Understand the mathematical theory.
2. Implement the concept in R (when provided by the course).
3. Reimplement the same concept in Python.
4. Run simulations and numerical experiments.
5. Connect the concept to quantitative finance applications.

This repository serves as both a learning journal and a long-term reference for quantitative finance, machine learning, and financial engineering topics.
